# HR-Data-Analysis-project
HR-Data-Analysis-Project---SQL-Power-BI
Performed data cleaning and preprocessing using SQL to ensure data accuracy and consistency then analyzed employee data to identify trends in salary, performance, and attrition. Developed 4 interactive dashboards in Power BI HR Overview, Salary Analysis, Performance Analysis, Attrition Analysis.

# Tools Used:
SQL (Data Cleaning & Analysis)

Power BI (Interactive Dashboards)

# Key Dashboards:
HR Overview: Employee distribution, demographics, and key KPIs

Salary Analysis: Salary trends across departments and job roles

Performance Analysis: Factors affecting employee performance

Attrition Analysis: Understanding employee resignation patterns

# Key Insights:
Identified variations in salary distribution across departments

Observed relationships between employee satisfaction and attrition

Highlighted top-performing employees and key performance drivers
